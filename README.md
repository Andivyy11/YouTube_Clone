# YouTube Clone
![HomePage](https://imgur.com/a/sifVMUD)

## A Video Sharing Web App
### Client - React , Styled-Componenets , Redux (State Management)
### Server - JavaScript , MongoDB

## Features 
## User Authentication
- Sign Up
- Sign In
- Google Sign In
- Log Out
- JWT Token

## Firebase Storage
- Upload thumbnail and video
- Download thumbnail and video
  
## Video Features
- Upload a video
- View trending and your subscription videos
- Like/Dislike videos
- Comment on a video
- Delete your comment
- Subscribe/Unsubscribe a Channel
- Play/Pause a Video
- View Your Channel
- Delete your video

## Additional Features
- Light/Dark Mode
